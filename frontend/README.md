# Frontend

React + TypeScript + Vite admin UI with:
- Tailwind
- shadcn-style UI components
- Zustand auth store
- Axios API wrapper
- React Router guards
- Dashboard, Files, Services, Docker, and Cron pages

## Run

1. `npm install`
2. `npm run dev`

Optional env:
- `VITE_API_BASE_URL` (default `http://127.0.0.1:8080`)
