# Security Policy

## Supported Versions

The project is currently under active development.
Security fixes are applied on the `main` branch.

## Reporting A Vulnerability

Please do not disclose vulnerabilities in public issues or pull requests.

Use GitHub Security Advisories to report privately:
https://github.com/snowfallx-bot/SnowPanel/security/advisories/new

Please include:

- A clear description of the issue
- Reproduction steps or proof-of-concept
- Potential impact
- Suggested mitigation (if available)

We will triage as quickly as possible and coordinate remediation.
